# 🐍 Snake MVC - PySide6

Minijuego clásico de Snake desarrollado en **Python** utilizando **PySide6** y siguiendo el patrón de diseño **MVC (Model - View - Controller)**.

---

# 📸 Características

✅ Interfaz gráfica moderna  
✅ Cuadrícula visual  
✅ Sistema de puntuación  
✅ Reinicio de partida  
✅ Movimiento fluido  
✅ Colisiones con paredes y cuerpo  
✅ Arquitectura MVC separada  
✅ Código sencillo y fácil de ampliar  

---

# 🛠️ Tecnologías utilizadas

- Python 3
- PySide6
- Programación orientada a objetos
- Patrón MVC
